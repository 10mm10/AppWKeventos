# WK Eventos - App de Orçamentos

Este é um aplicativo desenvolvido em React Native para gestão de orçamentos da WK Eventos.

## Tecnologias
- React Native / Expo
- Expo Print (Geração de PDFs)
- EAS Build (Android)

## Como rodar localmente
1. Clone o repositório
2. Execute `npm install`
3. Execute `npx expo start`
